Primeras fórmulas \( \Leftarrow \)

¡Atención!

Para poder empezar a escribir fórmulas necesitas haber incluido el código de MathJax en la cabecera de tu eXe. Si todavía no lo has hecho, hazlo ahora en la sección: Motor MathJax en eXe

En esta sección

Veremos cómo empezar con la creación de fórmulas sencillas en el cuerpo del texto de un documento de eXe. Para el funcionamiento de eXeLearning como entorno de trabajo en general puede consultarse: eXeLearning 2.1. Tutorial - Manual

Escribir fórmulas en el área de trabajo

Las fórmulas se escriben directamente en el texto del área de trabajo y no debe utilizarse el botón que lleva el editor. Mientras escribimos el texto y las fórmulas en el área de trabajo, estamos en modo edición, en estos momentos no podemos ver cómo queda nuestra fórmula, ni si está bien hecha, ya que lo que vemos es el código que escribimos. Para poder visualizarla correctamente deberemos pulsar el botón Hecho que tenemos en la parte inferior izquierda del editor: . Así pues, deberemos ir alternando entre el modo edición y modo de visualización del texto.

Hay dos modos para la visualización de fórmulas: texto en línea y modo resaltado.

Modo texto

Las fórmulas se escriben en línea con el texto y su tamaño se reduce en ocasiones para que se adapte mejor. Si deseamos fórmulas de este tipo encerraremos el código LaTeX con los símbolos de barra invertida y paréntesis: \( ... \). 

Por ejemplo:

  • \( ax^2 + bx + c = 0 \), da como resultado:  \( ax^2 + bx + c = 0 \) (el uso de espacios no afecta a la fórmula, se incluyen para aumentar su legibilidad). Cuando el tamaño del modo texto no nos satisfaga podemos forzar el modo resaltado anteponiendo \displaystyle. Véase, más abajo, el punto: Fracciones.

Modo resaltado

  • En el segundo de los casos la fórmula se escribirá en un párrafo aparte con alineación izquierda. En este caso pondremos dos símbolos de corchete a cada lado de la fórmula, de este modo: \[ ... \]. 

Por ejemplo: 

  •  \[ ax^2 + bx + c = 0 \], produce como resultado: \[ ax^2 + bx + c = 0 \]

En el modo resaltado las fórmulas quedan siempre en el margen izquierdo, para que queden centradas podemos usar tablas (aquí se explica cómo).

Subíndices y superíndices

  • Los subíndices se indican mediante una barra de subrayado. Si el subíndice tiene más de un carácter se encierran entre llaves: \( x_n, x_{n+1} \), produce \(x_n, x_{n+1}\)
  • Los superíndices se indican mediante una acento circunflejo. Si el superíndice tiene más de un carácter se encierran entre llaves: \( x^n, x^{n+1} \), produce \(x^n, x^{n+1}\)

Fracciones

Las fracciones se escriben con el código \frac {numerador} {denominador}. Si nuestra fracción consta de un único carácter en el numerador y otro en el denominador, podremos omitir las llaves. Las llaves se usan siempre para agrupar términos y no salen en la ecuación. Si la fracción se hace en modo texto siempre queda de un tamaño más pequeño, si la deseamos más grande utilizaremos \displaystyle justo delante de \frac, también es posible utilizar \dfrac que tiene el mismo efecto.

  • Modo texto por defecto: \( f(x) = \frac {x+1} {x^2} \): \( f(x) = \frac {x+1} {x^2} \)
  • Modo texto forzando el modo resaltado:
    • \( f(x) = \displaystyle \frac {x+1} {x^2} \): \( f(x) = \displaystyle \frac {x+1} {x^2} \)
    • \( f(x) = \dfrac {x+1} {x^2} \): \( f(x) = \dfrac {x+1} {x^2} \)

Paréntesis

Para que la fracción, o cualquier otra expresión, quede encerrada entre paréntesis grandes pondremos \left ( ... \right ) al principio y final de la expresión . Esto se aplica también a las llaves y los corchetes. Véase la sección Delimitadores para conocer más opciones.

  • \( \left ( \displaystyle \frac {x+1} {x^2} \right ) ^2 \): \( \left ( \displaystyle \frac {x+1} {x^2} \right ) ^2 \)
