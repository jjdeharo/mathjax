Motor MathJax en eXe

En esta sección

Se describe la forma de incorporar el motor de MathJax en nuestro documento de eXe, de forma que quede disponible para poder empezar a escribir nuestras fórmulas de forma inmediata.

Damos por supuesto que tienes instalado eXeLearning en tu ordenador y que eres usuario del mismo. Si no es el caso no hay ningún problema, desde aquí lo puedes descargar, en cuanto a su uso es bastante intuitivo y aquí tienes el manual de eXe.

Para poder disponer de toda la potencia de \(\LaTeX\) en eXeLearning bastará abrir un documento de eXe ya existente o crear uno nuevo y añadir en Propiedades > Opciones avanzadas > Personalizar la sección HEAD, el texto que tienes en el cuadro azul. Para ello copia el código que hay a continuación y pégalo en la sección HEAD indicada antes.

<script type="text/x-mathjax-config">
  MathJax.Hub.Config({
    extensions: ["tex2jax.js"],
    jax: ["input/TeX", "output/HTML-CSS"],
    tex2jax: {
      inlineMath: [ ['$','$'], ["\\(","\\)"] ],
      displayMath: [ ['$$','$$'], ["\\[","\\]"] ],
      processEscapes: true
    },
    "HTML-CSS": { fonts: ["TeX"] }
  });
</script>

<script type="text/x-mathjax-config">
MathJax.Hub.Config({
  TeX: { equationNumbers: { autoNumber: "AMS" } }
});
</script>

<script type="text/javascript"
   src="https://cdnjs.cloudflare.com/ajax/libs/mathjax/2.7.5/latest.js?config=default">
</script>
